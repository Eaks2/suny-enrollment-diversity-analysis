# SUNY Enrollment Diversity Analysis

Statistical analysis of SUNY enrollment diversity trends using R and Excel, with data cleaning, demographic trend analysis, ANOVA, Tukey HSD, and visual reporting.

## Overview

This project analyzes long-term SUNY enrollment data to identify demographic trends and statistically significant differences across student groups. The project focused on cleaning and structuring historical enrollment data, reviewing demographic categories, conducting statistical testing, and presenting findings through charts and summary visuals.

## Tools Used

- R
- Excel
- ANOVA
- Tukey HSD
- Data Cleaning
- Data Visualization

## Key Features

- Cleaned and structured 20+ years of enrollment data for demographic trend analysis
- Reviewed missing values, inconsistent categories, and long-term enrollment patterns
- Conducted ANOVA and Tukey HSD testing to compare enrollment differences across demographic groups
- Created charts and summary visuals to communicate findings clearly
- Framed results around institutional reporting, equity trends, and data-informed decision-making

## Repository Contents

- `analysis_code.R` – R code used for data cleaning, statistical testing, and visualization
- `final_report.pdf` – Full project report with methodology, results, charts, and findings
