# SUNY Enrollment Diversity Analysis

Statistical analysis of SUNY enrollment diversity trends using R and Excel, with data cleaning, demographic trend analysis, ANOVA, Tukey HSD, and visual reporting.

## Overview

This project analyzes long-term SUNY enrollment data to identify demographic trends and statistically significant differences across student groups. The project focused on cleaning and structuring historical enrollment data, reviewing demographic categories, conducting ANOVA and Tukey HSD testing, and presenting findings through charts and summary visuals.

## Files

- `final_report.pdf` – Full project report with methodology, statistical analysis, visualizations, and findings.
